# Trezoa SDK WASM JS

Use the Trezoa SDK WASM JS crate to build a JS package with `wasm-pack` usable in Node or browser environments.
See the [Trezoa Program Crate](https://crates.io/crates/trezoa-program) instead for on-chain programs, and, the [Trezoa SDK Crate](https://crates.io/crates/trezoa-sdk) for client-side
applications.

More information about Trezoa is available in the [Trezoa documentation](https://trezoa.com/docs).
